# Prueba Técnica ACEMA

## Pasos para correr el proyecto en local:

### 1. Clonar repositorio

```bash
git clone https://github.com/frezdev/acema-technical-test.git
```

Ó con SSH

```bash
git clone git@github.com:frezdev/acema-technical-test.git
```

### 2. Instalar Depencencias

```bash
npm install
```

### 3. Ejecutar comando de construcción de la app

```bash
npm run build
```

### 4. Iniciar Servidor local

```bash
npm run preview
```

### 5. Abrir la url local

[http://localhost:4173](http://localhost:4173)
